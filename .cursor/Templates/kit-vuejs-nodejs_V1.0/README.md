# Blueprint VueJS/NodeJS

Estrutura reutilizável para projetos web com VueJS + NodeJS tooling.

## Pastas canônicas
- `src/components`
- `src/composables`
- `src/router`
- `src/stores`
- `src/views`
- `src/services`
- `src/utils`

## Regras mínimas
- Composition API + `<script setup>`.
- Estado global em Pinia.
- Endpoints centralizados em `src/config.js`.
- Build e validação com `npm run dev` e `npm run build`.
