## Convenção de versionamento

- **Major (X):** mudança arquitetural ou quebra de compatibilidade.
- **Minor (y):** nova funcionalidade ou módulo sem quebra.
- **Patch:** apenas correções; registrar no changelog do arquivo afetado (`Changelog (este arquivo)`), não aqui.
- **Formato de data:** DD/MM/AAAA.
