# quick_ref — version-master-orchestrator

| Skill | Quando usar |
|-------|-------------|
| `version-semver-product` | calcular bump de versão (major/minor/patch) |
| `version-breaking-change-guard` | verificar breaking changes antes de refactor público |
| `version-deprecation-policy` | marcar feature/API como deprecated com prazo e alternativa |
| `version-migration-assistant` | documentar guia de upgrade entre versões |
| `version-release-notes` | gerar notas de release / CHANGELOG estruturado |

**Sequência release:** breaking-change-guard → semver-product → deprecation-policy → migration-assistant → release-notes
